# Моделирование распространения оптического вихря методом углового спектра

## Описание проекта

**Репозиторий** содержит реализацию численного моделирования распространения оптических вихрей в свободном пространстве с использованием метода углового спектра и преобразования Ханкеля. Проект демонстрирует подход, альтернативный традиционному преобразованию Фурье для работы со световыми полями вида $\PSI(\rho, \varphi) = w(\rho)\exp(im\varphi)$.

## Основные компоненты

* Реализация метода углового спектра с использованием **преобразования Ханкеля**
* Алгоритм преобразования одномерного массива в двумерное радиально-симметричное распределение
* Численный эксперимент с оптическим вихрем

## Требования

* Python 3.8+
* NumPy
* SciPy
* PyHank (библиотека для преобразования Ханкеля)
* Matplotlib

## Структура проекта

```
├── image_transforms/  # Модуль для преобразований изображений и полей
│   ├── __init__.py    # Инициализация пакета
│   ├── asm.py         # Реализация классического метода углового спектра (ASM)
│   ├── hasm.py        # Реализация метода углового спектра с преобразованием Ханкеля (HASM)
│   └── radial_to_cartesian.py  # Преобразование одномерного массива в двумерное распределение
├── modes/             # Модуль для работы с оптическими модами
│   ├── __init__.py    # Инициализация пакета
│   └── LG.py          # Реализация мод Лагерра-Гаусса
├── .gitignore
├── main.py            # Основной скрипт для запуска моделирования
├── README.md          # Документация проекта
└── optical-vortex-hankel-asm.pdf  # Развернутое теоретическое описание метода и математическое обоснование
```
